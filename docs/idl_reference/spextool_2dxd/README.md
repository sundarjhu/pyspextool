# Spextool IDL 2DXD reference files

These IDL source files are included temporarily as reference material for
reverse-engineering the iSHELL 2DXD wavelength-calibration / rectification
algorithm.

They are not part of the Python runtime and are not executed by pyspextool.

They come from the Spextool distribution and are being used for private
algorithm analysis only.

Purpose:
- inspect 1DXD → 2DXD control flow
- identify 2-D line tracing logic
- identify polynomial/distortion fitting
- identify rectification-index generation
